Автоматизация
Инструкции по подключению уведомлений и интеграций
Email-уведомления (Formspree)
При новой заявке — мгновенное письмо на ваш Gmail. Настройка за 5 минут, без кода.
  • Зарегистрируйтесь на formspree.io
  • Создайте форму → получите endpoint URL
  • В HTML форме: action="https://formspree.io/f/YOUR_ID"
  • Formspree пришлёт email при каждой заявке
Telegram-уведомления (бот)
Новая заявка → мгновенное сообщение в Telegram. Настройка через @BotFather.
  • Откройте @BotFather → /newbot → получите TOKEN
  • Узнайте ваш chat_id через @userinfobot
  • Используйте код ниже в вашем сервере или Make.com
  • POST https://api.telegram.org/bot{TOKEN}/sendMessage
Google Sheets (Make.com)
Каждая заявка автоматически добавляется строкой в Google Таблицу — ваша онлайн CRM.
  • Создайте сценарий на make.com (бесплатно)
  • Триггер: Webhook (вставьте URL в форму сайта)
  • Действие: Google Sheets → Add Row
  • Колонки: Имя, Email, Специальность, Дата, Статус
Полная автоматизация (n8n)
Цепочка: заявка → Telegram → Google Sheets → Email с ссылкой оплаты. Без кода.
  • Разверните n8n (cloud.n8n.io) или self-hosted
  • Webhook → разветвление на 3 действия
  • Telegram Bot: уведомление о новом лиде
  • Gmail: письмо с Kaspi Pay ссылкой участнику
Пример: отправка заявки в Telegram
fetch('https://api.telegram.org/botYOUR_BOT_TOKEN/sendMessage', {
  method: 'POST',
  headers: {'Content-Type':'application/json'},
  body: JSON.stringify({
    chat_id: 'YOUR_CHAT_ID',
    text: `🔔 Новая заявка!\nИмя: ${name}\nEmail: ${email}\nСпец: ${spec}\nДата: ${new Date().toLocaleString('ru')}`,
    parse_mode: 'HTML'
  })
});
